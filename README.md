# 萬福街96號 — 住戶公告網頁

此專案為「楊梅區萬福街96號 — 住戶公告版」的單一 HTML 頁面，提供：
- 公約與公告內容
- Wi‑Fi 資訊（可一鍵複製）
- 房東聯絡方式（可複製）
- 繳租銀行帳戶（可一鍵複製帳號）
- 產生本頁連結的 QR Code（便於列印張貼）

## 使用方式
1. 開啟 `index.html`，將頁面中的下列欄位改為您的實際資訊：
   - `【請填 SSID】`、`【請填 Wi‑Fi 密碼】`
   - `【請填姓名】`、`【請填手機 / 市話】`、`【若有請填】`
   - `【請填銀行名稱或代號】`、`【請填分行（非必填）】`、`【請填戶名】`、`【請填帳號】`
   - 頁面最上方的「公告日期」

2. 直接把 `index.html` 傳給房客，或放到任何可以公開的網址。

## 產生 QR Code
- 頁面下方提供「產生 QR Code」按鈕，會將目前頁面網址轉為 QR 圖片。
- 若是本機檔案（未上傳），QR 內容會是本機路徑，建議上傳到能公開存取的網址（例如雲端空間分享連結、GitHub Pages、個人網站）後，再產生 QR。
- 產生後可按「下載 QR 圖片」另存成 PNG 檔，列印張貼於公共區域。

## 部署（選擇其一）
- 直接用雲端硬碟分享連結（例如 Google Drive 設為任何人可讀取）
- 放進個人網域/主機（任何靜態空間皆可）
- GitHub Pages：建立公開 repo，將 `index.html` 放入，於 Repo 設定啟用 Pages

## 其他
- 複製功能使用 `navigator.clipboard`，在 HTTPS 或本機檔案也可運作（部分舊瀏覽器會使用備援方式）。
- 生成 QR 功能會即時載入 CDN：`qrcodejs`，需要可連外網路才能載入函式庫。
